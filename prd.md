# Product Requirements Document (PRD)

## 1. 제품 개요 (Overview)

### 제품명

**CLIMATE SWITCH : 경기도 기후 시뮬레이터**

### 한 줄 설명

경기 기후위성 데이터와 공간정보(지도)를 활용하여, 도시 요소(녹지·불투수면·쉼터 등)를 ON/OFF 했을 때 폭염·침수 위험이 어떻게 변하는지 **즉시 시각화하고 정량화하는 체험형 지도 서비스**.

---

## 2. 배경 및 문제 정의 (Problem Statement)

* 기후 데이터(폭염, 침수, 위성자료)는 존재하지만 일반 시민과 정책 담당자가 **직관적으로 이해하기 어렵다**.
* 기존 지도 서비스는 “현재 상태”만 보여주며,
  **도시 구조 변화가 기후 위험에 미치는 영향**을 체험할 수 없다.
* 정책·시설(공원, 투수면, 쉼터)의 필요성을 시민에게 설득하기 어렵다.

---

## 3. 목표 (Goals)

### 핵심 목표

1. 기후 데이터를 **정보 → 실험 → 이해**의 단계로 확장
2. 지도 위에서 “만약 ○○가 없었다면?”이라는 질문에 답 제공
3. 기후 정책·시설의 효과를 **시각적·수치적으로 설명**

### 해커톤 목표

* 경기 기후위성데이터 + 공간정보를 **실질적으로 활용**
* 창의성·실용성·데모 임팩트 모두 확보

---

## 5. 핵심 사용자 시나리오 (User Scenarios)

### 시나리오 1: 폭염 실험

1. 사용자가 동네 선택
2. “🌳 녹지 감소” 스위치 ON
3. 지도 색상이 붉어지며 폭염 위험 증가 표시
4. “폭염 위험 +52%” 수치 확인

### 시나리오 2: 침수 실험

1. 침수 모드 선택
2. “🧱 불투수면 증가” 슬라이더 조작
3. 침수 위험 구역 확대
4. 침수 취약 시설 강조 표시

## 6. 핵심 기능 요구사항 (Core Features)

### 6.1 지도 시각화

* WMS/WMTS 기반 레이어 렌더링
* 폭염 / 침수 모드 전환
* 지도 클릭 시 해당 위치 요약 정보 제공

### 6.2 시뮬레이션 스위치 패널

* 🌳 녹지 감소 (%) 슬라이더
* 🧱 불투수면 증가 (%) 슬라이더
* 🏠 무더위쉼터 ON/OFF
* 🌊 침수 방어 ON/OFF

### 6.3 Before / After 비교

* 위험 지수 전/후 비교
* 증가율(%) 표시
* 위험 구역 변화 시각화

### 6.4 결과 리포트

* 폭염 위험 지수 변화
* 침수 위험 면적 변화
* 가장 위험 증가 구역 Top 3
* 자동 생성 인사이트 문장

## 7. 데이터 요구사항 (Data Requirements)

### 사용 데이터 (경기기후플랫폼 API)

#### 폭염 관련

* 폭염 체감온도 / 열환경지도
* 경기도 열쾌적성 평가
* 녹지 현황도
* 투수·불투수 유형 지도

#### 침수 관련

* 침수흔적지도
* 극한호우 위험도 지수
* 하천 / 소하천

#### 안전시설

* 무더위쉼터 POI
* 응급·의료시설 (선택)

---

## 8. 시뮬레이션 로직 (Simulation Logic)

### 폭염 위험 점수 (0~1)

```
HeatRisk = 0.55*HeatIndex + 0.25*Impervious - 0.20*Green
```

시뮬레이션 반영:

* 녹지 감소 → 위험 증가
* 불투수면 증가 → 위험 증가
* 쉼터 OFF → 취약도 증가

### 침수 위험 점수 (0~1)

```
FloodRisk = 0.40*RainRisk + 0.25*FloodTrace + 0.20*Impervious + 0.15*RiverProximity
```
